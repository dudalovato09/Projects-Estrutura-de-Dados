6. O que há de errado com o seguinte trecho de código?

char *a, *b;
a = "abacate";
b = "uva";
if (a < b)
   printf ("%s vem antes de %s no dicionário", a, b);
else
   printf ("%s vem depois de %s no dicionário", a, b); 

Resposta: o que está errado é a forma de comaração, ele usa o (a > b) para comparar duas variáveis do tipo char o correto deveria ser "if (strcmp(a, b) < 0)"

O "if (strcmp(a, b) < 0)" iria retornar 

- Valor < 0 se a vem antes de b no dicionário.
- 0 se são iguais.
- Valor > 0 se a vem depois de b.
